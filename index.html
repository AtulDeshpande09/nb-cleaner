🔧 Common Issues This Tool Fixes

If you're seeing these errors on GitHub, Notebook Cleaner can help:

  • "Sorry, something went wrong. Reload?" — intermittent notebook rendering failures
    Often caused by complex widget metadata or large outputs. Cleaning the file resolves the renderer conflict [[1]].
  • "state key missing from metadata.widgets" — notebooks from Colab/JupyterLab not displaying
    Removing the widgets block from metadata restores rendering [[9]]. Our tool does this automatically.

Tip: After cleaning, re-upload your notebook to GitHub. Rendering usually updates within 1-2 minutes.
